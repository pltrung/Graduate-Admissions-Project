Data Introduction¶

Graduate Admissions dataset consists of 9 columns with 400 samples in total. The features of this dataset can be found here:

  • GRE Scores (290 to 340)
  • TOEFL Scores (92 to 120)
  • University Rating (1 to 5)
  • Statement of Purpose Rating(1 to 5) (SOP)
  • Letter of Recommendation Strength (1 to 5) (LOR)
  • Undergraduate CGPA (6.8 to 9.92)
  • Research Experience (0 or 1)
  • Chance of Admit (0.34 to 0.97)

My aim and purpose for this dataset is to predict whether a student will get admitted to a Master's degree given the features of the dataset.

The full dataset can be found at https://www.kaggle.com/mohansacharya/graduate-admissions.

Preparing for Data Mining¶

Since variable Serial No. is an index and unique key identifier, we will not be using this variable in our analysis. Our predictor variable is Chance of Admit, since we want to know if a student have a good chance of admitting into a Master's degree or not. We would also need to normalize our variables since diSifferent variables are on different scales, so normalizing all the variables to fit into one scale helps us understand our model better.

The model I am building for this particular project is a classification model, meaning that our predictor variable should be binary. Right now, it is not a binary factor, and we need to build a threshold of our choosing in order to split the variable into binary.

I have used 75% for Chance of Admit as my threshold, therefore all samples that has Chance of Admit greater than 0.75 will be given the 1 label and all variables below that threshold will be given a 0 label. This means that when we do our analysis, a label of 1 means that the student will be able to get into the Master's degree, meanwhile 0 means that the student will not.

We will also be using 80/20 for our data partition.

Conclusion¶

After exploring with the dataset and using two methods of classification regression models, the random forest classifier model proves to perform better at predicting whether a student will get admitted into a Master's degree or not. There were a lot of assumptions made in the dataset as well as in this project, if given more time, we could gather more data after the student got into the Master's degree, since the variable Chance of Admit is a probability variable and not a binary variable. We could also use regression algorithms to predict the probability of Chance of Admit.
